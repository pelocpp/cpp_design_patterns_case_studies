# Passwortcheck

#### Aufgabe

Schreiben Sie ein Programm, das die Passwortstärke anhand vordefinierter Regeln überprüft.
Diese Regeln sollen in verschiedenen Kombinationen ausgewählt werden können.

Die Minimalanforderung an ein Passwort besteht darin, das es eine Mindestlänge von 8 Zeichen aufweist.

Darüber hinaus können zusätzliche Regeln verlangt werden, wie zum Beispiel
das Vorhandensein eines Sonderzeichens, einer Ziffer oder eines Groß- und Kleinbuchstabens.

Schreiben Sie eine Testfunktion, die ein oder mehrere Regeln anwendet.
